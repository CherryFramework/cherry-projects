## Features
* CPT Portfolio
* Page template (named Portfolio) + single template
* Shortcode (named cherry_portfolio)
* 20+ custom hooks
* Translation (Localization)
* Ajax loading content

## How to use

#### In a post/page
Insert a shortcode `[cherry_portfolio]` to the post/page content.
